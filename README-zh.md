# Webpage HTML Export Pro

[English](README.md) | [中文](README-zh.md)

本插件基于 https://github.com/KosmosisDire/obsidian-webpage-export 二开，增加、优化了许多功能，使其更适合作为博客使用
## 使用方法
1. 安装插件
2. 在插件设置中填好您的一些基本配置、开启你喜欢的功能
3. 按下 `ctrl+p` 选中`set html export settings` 
4. 选中你想要导出的笔记，点击导出即可

## 新增功能
请看这里：
https://myblog-livid-iota.vercel.app/00-home/blog/webpage-html-export-pro.html

**本插件实现了i18n、请不要担心语言问题**
- **代码块增强**：支持一键复制、自动换行、默认折叠/展开以及清晰的语言标识。
- **代码行高亮**：支持通过 `>>>> ` 语法实现代码行背景高亮，提升代码可读性。
- **交互式图片查看**：支持点击图片进入缩放模式，并可通过鼠标滚动进行实时缩放。
- **长图自动折叠**：针对长度过大的图片会自动折叠处理，避免页面过长，保持排版精美。
- **YAML 属性显示**：可在文章标题下方直接显示笔记属性，支持自定义默认显示状态。
- **智能大纲视图**：支持根据当前阅读进度自动高亮对应标题，并可设置标题的默认折叠层级。
- **文件夹数量统计**：在侧边栏文件树中实时显示每个文件夹下的笔记总数。
- **附件下载支持**：支持直接从导出的网页中下载笔记关联的附件文件。
- **标准脚注渲染**：完美支持并渲染 Obsidian 标准的脚注语法。
- **Giscus 评论集成**：内置评论区功能，只需填写脚本代码即可自动解析加载。
- **隐私页面加密**：支持通过属性对特定笔记设置访问密码，实现精准的内容权限控制。
- **页脚内容定制**：可自由配置版权声明信息及底部的自定义功能链接。
- **全局导航栏**：支持置顶导航菜单，方便快速跳转至个人介绍、日志或其他笔记页面。
- **全文模态搜索**：支持 `Ctrl+K` 快速唤起搜索，并能通过 `Tab` 键预览内容的上下文摘要。
- **文档时间戳显示**：在文章显著位置展示笔记的创建时间与最后更新日期。
- **Vercel Insights 支持**：集成 Vercel 部署分析功能，方便跟踪流量与性能数据。
- **配置持久化管理**：支持保存、更新及切换多套导出配置方案，满足不同的导出需求。
